# What is it ?
Its a simple """""blog"""" app created using MERN stack => MongoDB, Express.js, React.js and Node.js 
Everything writen in Typescript

## Users can:
- Register, Login
- Create new posts with pictures
- Delete posts 
- Look at posts of other users
